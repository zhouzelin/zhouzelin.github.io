# 允许拖拽 draggable

绑定该指令，实现元素拖拽功能。给父元素添加position: relative的样式，被绑定的元素就可以在父元素内拖拽移动。

# 正常用法

<template>
  <div class="container">
    <div class="box" v-draggable></div>
  </div>
</template>

<style scoped>
.container {
  height: 400px;
  width: 400px;
  border: 1px solid #ddd;
}
.container .box {
  height: 20px;
  width: 20px;
  background-color: red;
}
</style>

# 事件回调

事件名称 说明 回调参数
drag-start 拖拽开始。 event
drag 拖拽。 event
drag-end 拖拽结束。 event
